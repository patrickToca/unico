About Unico Enabling it, disabling, and more

Unico is an coding experiment, not a real product. Feel free to use it if you will, but be warned that as an experiment, it can be terminated at any moment. Also, changes can occur without prior notice that affect how it works. USE IT AT YOUR OWN RISK.

Hi there! A few more points to have in mind before you use Unico:

  • No passwords are stored. Everything uses OAuth.
  • Unico was created to scratch a particular itch for its author and even though Unico is not a secret, I'm also not trying to actively spread the word about it. Please don't go posting about Unico, because if too many people sign up, I may have to implement use controls and I really don't want to do that. :)
  • Due to limits in the Google+ API, Unico is not able to access posts shared with Circles. Only public posts are available.
  • If you have problems or suggestions, use the Contact link above. Enjoy it ;)
